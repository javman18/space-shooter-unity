Space Shooter (Prueba Técnica Unity)

Shooter espacial vertical infinito.

Controles
- Movimiento: WASD
- Disparo: Click izquierdo
- Pausa: ESC

Características
- Estados: Menú / Jugando / Pausa / Game Over
- UI: vida y score (responsiva)
- Game feel: feedback de daño/muerte, post-processing (Bloom, etc.)
- Loop infinito con spawns de enemigos

Gestión de memoria
- Object Pooling con `PoolService` + `IPoolable` (prewarm y reutilización)
- Pool usado para balas / enemigos / VFX

Build
- Windows x64 en Releases: `v1.0`

Unity
- Versión: 6000.0.43f1

Créditos
- Sound effects de Freesound.org y pixabay
- Dynamic Space Background Lite - DinV Studio
- Free Stylized 2D Space Shooter Pack - Larzes
- UI de itch.io
